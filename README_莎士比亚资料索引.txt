资料内容概要：
================================================================

一、生平传记
  - 中文详细生平（含家庭、婚姻、创作生涯、晚年、年表）
  - 英文完整传记（English biography with timeline）

二、作品简介

  - 长篇叙事诗介绍
  - 完整作品列表（含创作年代）

四、图片资料

  - 高清图片下载链接

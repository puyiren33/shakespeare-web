# 🎭 莎翁奇妙物语 — 莎士比亚互动之旅

一个以莎士比亚生平与作品为主题的交互式单页 HTML 游戏。融合角色养成、剧情抉择、收集图鉴、迷你游戏与沉浸式作品穿越体验。

---

## 项目创意构思

### 核心理念
将莎士比亚的生平传记与经典作品转化为可交互的游戏体验，让用户在「玩」的过程中自然地了解这位文学巨匠的人生轨迹与创作世界。

### 创意来源
- **人生之旅**：以时间线驱动剧情抉择，用户的选择影响声望/情感/财富/智慧四项属性，导向不同结局
- **莎翁世界**：11 位关键人物（家人、友人、对手、赞助人、神秘人）构成可收集的角色图鉴
- **戏中戏**：16 款迷你游戏覆盖不同知识点——年代排序、悲喜辨识、人物猜谜、冷知识、真假名言、角色归属、戏剧术语、十四行诗韵律、票房风云、字母解密，以及原有的哈姆雷特抉择、恋人对对碰、名言拼图、十四行诗填空、单词发明家、台词速配
- **作品穿越**：14 部莎翁名剧各有详细简介、名句赏析，点击「穿越」进入沉浸式交互场景，每部剧包含双关卡体验（游戏 + 剧情测验）
- **名句轮播**：经典双语名言循环展示

### 设计目标
- 零门槛：打开浏览器即可玩，无需安装任何依赖
- 沉浸感：深色星空背景 + SD 风格角色动画 + 动态粒子效果 + 金色复古 UI
- 教育性：知识点融入游戏机制，边玩边学

## 技术方案简述

| 层面 | 方案 |
|---|---|
| **架构** | 单页 HTML 应用，零构建工具，零外部依赖（除 Google Fonts） |
| **样式** | 纯 CSS3，包含自定义变量、渐变、弹性布局、网格布局、关键帧动画 |
| **图形** | Canvas 2D API 绘制 SD 风格角色（6 个年龄段，5 种表情，眨眼动画，待机呼吸） |
| **交互** | 原生 JavaScript (ES6)，自包含事件绑定，阴影 DOM 无依赖 |
| **数据** | JavaScript 对象/数组硬编码，含生平场景、角色资料、作品信息、游戏题库 |
| **动画** | CSS keyframes（渐入、脉冲、闪烁、波纹涟漪）+ Canvas（动态星空、流星、星云、浮动粒子） |
| **响应式** | CSS 媒体查询适配平板与手机 |

### 文件结构

```
莎士比亚资料大全/
├── 莎士比亚互动之旅.html   ← 唯一游戏文件（~1920 行，~100KB）
├── README.md                ← 本设计说明
└── 其他资料文件夹/          ← 离线资料（未纳入 Git）
```

## AI 协作过程概述

本项目的全部代码由 **AI 辅助编程工具 (opencode)** 与人类开发者实时结对编写，使用 **big-pickle** 大模型驱动。协作流程如下：

### 各轮次演进

| 轮次 | 需求 | 实现内容 |
|---|---|---|
| 1 | 初始搭建 | Canvas 角色绘制 + 5 标签页布局 + 剧情抉择系统 + 属性统计 + 世界收集 + 作品卡片 + 猜剧名测验 + 3 款初始小游戏 |
| 2 | 增加互动 | 新增 3 款小游戏（十四行诗填空、单词发明家、台词速配） |
| 3 | 充实作品 | 14 部剧详情页（长简介 + 角色 + 名句赏析）+ 穿越互动系统（3 种游戏类型）+ 第二关剧情测验 |
| 4 | 再丰富 | 所有简介扩充到 3-5 句历史细节 + 每剧名句与解读 + 双关卡穿越 |
| 5 | 再加游戏 | 批量新增 10 款小游戏，戏中戏总数达 16 款 |
| 6 | 动画升级 | 标题流光 / 逐层入场 / 呼吸辉光 / 卡片悬停 / 弹出动画 / 按钮涟漪 / 标签过渡 / 暗角效果 / 星空（200 星 + 星云 + 流星）/ 浮动粒子扩增 |
| 7 | 修复部署 | 修复 HTML 实体渲染（textContent → innerHTML）+ 初始化 Git + 推送 GitHub |

### 协作模式

```
用户 → 提出需求（中文自然语言）
  ↓
AI   → 阅读现有代码 → 规划实现方案 → 执行文件编辑
  ↓
用户 → 反馈验收 / 提出修改
  ↓
AI   → 调整并验证（JS 语法检查 + 编码检查）
```

整个过程迭代 7 轮，每个需求平均 2-5 分钟完成从理解到实现。
